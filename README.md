# Welcome to Degen DKG Gaming Solidity Contract!

Hello there, welcome to my project! For this project, I have written a smart contract that implements the mint(), balanceOf(), transfer(), and all other functions of the ERC20 token standard.


## Description 

The contract is an implementation of the ERC20 token standard, with additional features such as minting and burning of tokens. The contract is designed to be owned by a single entity, who has the exclusive right to mint new tokens.
